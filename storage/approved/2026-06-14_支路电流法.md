一、支路电流法

1. 定义与步骤

以各支路电流为未知量，直接应用**基尔霍夫电流定律kcl**和**基尔霍夫电压定律kvl**列出独立方程，联立求解。

标准步骤（b 条支路，n 个节点）：

1. 选定各支路电流参考方向。
2. 根据**基尔霍夫电流定律**列出 n-1 个独立节点电流方程。
3. 根据**基尔霍夫电压定律**列出 b - (n-1) 个独立回路电压方程（通常选网孔或单连支回路）。
4. 联立求解方程组，得到各支路电流。
5. 如需电压或功率，再根据欧姆定律计算。

2. 关键特性

· 适用于任何线性电路。
· 方程个数 = 支路数 b
· 需注意电流源支路处理：若已知电流源电流，则该支路电流已知，可减少未知量。
· 网孔越多，方程越多

3. 典型例题

电路：Us1=12V，Us2=6V，R1=2Ω，R2=4Ω，R3=6Ω。求各支路电流。

```
        R1         a          R2
   ┌──2Ω──┬───────┬──4Ω──┐
   │         │           │         │
   Us1      R3          Us2        │
   12V      6Ω          6V         │
   │          │           │         │
   └──────┴───────┴──────┘
```

解：

· 支路数 b=3，节点数 n=2 → 1 个 KCL 方程，2 个 KVL 方程。
· 设电流 I1、I2、I3 方向如图（I1 从 Us1 正极流出，I2 从 Us2 正极流出，I3 从 a 流向地）。
· KCL（节点 a）： I_1 + I_2 = I_3   …(1)
· KVL（左回路，顺时针）： 12 - 2I_1 - 6I_3 = 0   …(2)
· KVL（右回路，顺时针）： 6 - 4I_2 - 6I_3 = 0   …(3)
    注意 Us2 极性：从负极到正极电势升 6V，按顺时针路径经过 4Ω 电阻压降 4I_2，再经过 6Ω 压降 6I_3，回到起点。
· 解方程：由(2)得  I_1 = 6 - 3I_3 ；
          由(3)得  I_2 = 1.5 - 1.5I_3 ；代入(1)：
     6 - 3I_3 + 1.5 - 1.5I_3 = I_3  →  7.5 - 4.5I_3 = I_3  →  7.5 = 5.5I_3  →  I_3 ≈ 1.3636A 
     I_1 = 6 - 3×1.3636 = 1.9092A ， I_2 = 1.5 - 1.5×1.3636 = -0.5454A （负号表示实际方向与假设相反）。

答案：I1≈1.91A，I2≈-0.55A（即从 a 流向 Us2 正极），I3≈1.36A。

4. 考试频率：★★★☆

· 基础计算题（含受控源时难度增加）
· 典型分值 8-12 分（中等题）

5. 难点与易错点

· 独立方程个数判断错误：KVL 回路必须独立（每个新回路至少含一条新支路）。
· 电流源支路处理：若已知电流源值，则该支路电流已知，可减少未知量，需灵活设变量。
· 方程符号错误：绕行方向与电压升降的符号（尤其含电动势时）易错。

二、戴维南定理

1. 定义与步骤

任何一个线性有源二端网络，对外电路而言，可以用一个理想电压源串联一个电阻的等效电路代替：

· 电压源电压  U_{oc}  = 网络开路时的端口电压；
· 串联电阻  R_{eq}  = 网络内部独立源置零（电压源短路、电流源开路）后的等效电阻。

解题步骤：

1. 断开待求支路（或负载），得到两个端口。
2. 求开路电压  U_{oc} （用节点法、网孔法等）。
3. 求等效电阻  R_{eq} ：
   · 独立源置零（电压源短路，电流源开路），用串并联或外加电源法。
4. 画出戴维南等效电路（ U_{oc}  与  R_{eq}  串联），接入待求支路，计算电流或电压。

2. 关键特性

· 适用于求某一支路的电压/电流/功率，避免解复杂方程组。
· 可以推广到被控制电源控制源电路（置零时受控源保留，用外加电源法求  R_{eq} ）。
· 最大功率传输定理：当负载电阻  R_L = R_{eq}  时，负载获得最大功率。

3. 典型例题

电路：Us=20V，R1=5Ω，R2=10Ω，R3=4Ω，R4=6Ω。求通过 R4 的电流。

解：

· 断开 R4，端口 a、b 为原 R4 两端。（峰明法）
· 求  U_{oc} ：此时 R1 与 R2 串联（10V 分压？先求总电流）。
    实际电路：Us 正极 → R1 → R2 → 回到负极；a 点接在 R1 与 R2 之间？我们设：R1 上端接 Us+，下端接 a；R2 上端接 a，下端接 Us-。则 a 点电压 = R2 分压。
    电流  I = \frac{20}{5+10}=1.333A ， U_{a}= I×R2 = 1.333×10=13.33V （对地）。
    b 点通过 R3 接地？若 b 点直接接地，则  U_{oc}=U_{ab}=13.33V 。
· 求  R_{eq} ：独立源置零（Us 短路），从 a、b 看进去：R1 与 R2 并联（因 Us 短路使 R1 下端与 R2 下端短接），再与 R3 串联？注意 b 点是否接地？若 b 直接接地，则 R3 一端接 b（地），另一端接？原电路 R3 一端接 Us-？需要具体拓扑。
    为简洁，假设正确化简后得  R_{eq}= (R1∥R2) + R3 = (5∥10)+4 = 3.333+4=7.333Ω 。
· 戴维南等效： U_{oc}=13.33V  串联  R_{eq}=7.333Ω ，接入 R4=6Ω，则
     I_{R4}= \frac{13.33}{7.333+6}= \frac{13.33}{13.333} ≈ 1A 。

答案：I≈1A（取决于实际电路，此处仅为方法演示）。

4. 考试频率：★★★★★

· 必考大题，常与最大功率传输结合
· 典型分值 10-15 分

5. 难点与易错点

· 受控源处理：置零时受控源不能置零，必须用外加电源法求  R_{eq} 。
· 开路电压求法：若网络复杂，需列方程，注意端口开路不影响内部。
· 等效电阻求反：独立源置零后，特别注意短路跨接对串并联关系的影响。
· 正负号：开路电压极性要与待求支路电流方向协调。

三、叠加定理

1. 定义与步骤

在线性电路中，多个独立源共同作用产生的响应（电压或电流），等于每个独立源单独作用时响应的代数和。

标准步骤：

1. 选定待求量（电压或电流）的参考方向。
2. 每次保留一个独立源，其余独立源置零（电压源短路，电流源开路）。
3. 计算该独立源单独作用时的响应值（注意符号：与参考方向一致为正）。
4. 重复步骤 2-3 对所有独立源。
5. 将所有独立源对应的响应值代数和（注意正负）。

2. 关键特性

· 仅适用于线性电路（电阻、线性受控源、独立源）。功率不满足叠加（需先求电压/电流再算功率）。
· 受控源始终保留（不能置零），视为电阻处理。
· 可简化多源电路分析，尤其适用于电源数目少（2~3 个）的情况。

3. 典型例题

电路：Us=12V，Is=2A，R1=2Ω，R2=4Ω，R3=6Ω。求流过 R3 的电流 I3（方向向下）。

```
      R1        a
   ┌──2Ω──┬───────┐
   │         │           │
  Us        R3           Is
  12V       6Ω          2A
   │          │           │
   └──────┴───────┘
         b
```

解：

· Us 单独作用，Is 开路：R1 与 R3 串联，总电阻 2+6=8Ω，电流  I_3' = \frac{12}{8}=1.5A （方向向下，为正）。
· Is 单独作用，Us 短路：Is 2A 分流。R1 与 R3 并联？注意：Us 短路后，R1 下端与 R3 下端短接，R1 上端与 R3 上端短接？实际：电流源 Is 接在 a 与 b 之间，Us 短路后，节点 a 经 R1 到 b，同时节点 a 经 R3 到 b，所以 R1 与 R3 并联。
    并联电阻  R_{eq}= \frac{2×6}{2+6}=1.5Ω ，Is 总电流 2A 流过并联部分，则通过 R3 的电流  I_3'' = 2×\frac{R1}{R1+R3} = 2×\frac{2}{8}=0.5A （方向？Is 电流从 a 流向 b，在并联电路中，电流倾向于走电阻小的支路，R1 小所以 R1 电流大，R3 电流方向也是从 a 到 b，即向下，故为正）。
· 叠加： I_3 = I_3' + I_3'' = 1.5 + 0.5 = 2.0A 。

答案：I3 = 2A（向下）。

4. 考试频率：★★★★☆

· 中高频题，常与戴维南结合考查
· 典型分值 8-12 分

5. 难点与易错点

· 受控源不能置零：叠加时受控源必须保留，否则会改变电路结构。
· 功率不叠加：常有题目陷阱问“总功率是否等于各电源单独作用功率之和”，答案是否。
· 正负号：每个分量的方向必须与原始参考方向比对，反向则取负。
· 独立源置零：电压源短路（一根导线），电流源开路（断开），切忌保留内阻。

四、三者对比（个人认为）

| 方法 | 适用场景 | 优点 | 缺点 |
| --- | --- | --- | --- |
| 支路电流法 | 支路数少（≤3）| 概念直接 | 方程多，计算量大 |
| 戴维南定理 | 求某一条支路 | 化繁为简，可求最大功率 | 需先求开路电压和等效电阻 |
